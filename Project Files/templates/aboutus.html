About Us

Learn about our mission, vision, and the team behind the Rice Type Classifier project.

Our Mission

Our mission is to revolutionize the agricultural industry by providing a cutting-edge solution for rice classification. We leverage artificial intelligence and machine learning to help farmers, distributors, and researchers accurately identify rice varieties with unprecedented speed and precision.

Our Vision

We envision a world where technology empowers every stakeholder in the agricultural supply chain. By making rice classification accessible and efficient, we aim to contribute to global food security and sustainable farming practices.

Tech Stack Used

HTML5 for structuring web pages
CSS3 for styling
Python 3.10 main programming language.
Flask for micro web framework for handling HTTP requests, routing, and rendering templates.
Transfer Learning for MobileNetV2 model from TensorFlow Hub used for extracting features.
OpenCV (cv2) for reading, resizing, and preprocessing images.
NumPy for numerical array manipulation.
TensorFlow for deep learning model handling (loading .h5 file, making predictions).
